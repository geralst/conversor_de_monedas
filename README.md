# Conversor de monedas
Challenge Conversor es una herramienta versátil que ofrece dos funciones principales: la conversión de monedas. Pensado para viajeros, estudiantes, ingenieros y cualquier persona que necesite realizar conversiones de forma rápida y precisa, su diseño intuitivo y amigable lo hace accesible para usuarios de todas las edades y niveles de experiencia.
## Acerca del Proyecto
Este proyecto fue desarrollado como parte del programa de formación en Java de ONE + Alura LATAM, una plataforma de aprendizaje en línea especializada en tecnología. El propósito del Challenge Conversor es aplicar y demostrar los conocimientos adquiridos en programación Java, a través de la creación de una herramienta práctica y funcional para la conversión de monedas.

## Características Principales
- Conversión de Monedas: Permite convertir entre monedas internacionales de forma sencilla.
- Interfaz de Usuario Intuitiva: Diseño limpio, moderno y fácil de navegar, que simplifica la selección de monedas o unidades de temperatura.
- Compatibilidad Multiplataforma: Al estar desarrollado en Java, puede ejecutarse en sistemas Windows, MacOS y Linux sin inconvenientes.

## ¿Como Utilizar el proyecto?
1. Clona este repositorio o descarga el archivo ZIP del proyecto.
2. Abre el proyecto en tu entorno de desarrollo Java preferido.
3. Compila y ejecuta la aplicación.
4. Sigue las instrucciones en pantalla para realizar las conversiones de moneda
